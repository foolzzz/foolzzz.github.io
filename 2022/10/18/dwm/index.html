alt + shift + enter 打开新窗口

ait + I 竖向排列
alt + Enter 窗口位置互换
alt + num 切换标签页
alt + shift + num 移动窗口至某标签页
alt + T 平铺
alt + M 单窗口模式
alt + F 浮动模式

alt + K

改变窗口的长度/比例
alt + H
alt + L

alt + space
alt + shift + space
